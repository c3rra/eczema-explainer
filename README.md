# Eczema: how it actually works

A plain-language explainer of the eczema mechanism, with the everyday version
and the technical version set side by side.

Covers the skin barrier, the immune overreaction, the shift in skin bacteria,
the itch and scratch cycle, and which treatment addresses which problem.

Single self-contained HTML page. No build step, no dependencies.

**Live:** https://c3rra.github.io/eczema-explainer/

Not a substitute for a GP or dermatologist.
